# Flight-Delay-prediction
Данный репозиторий содержит программную реализацию курсовой работы, посвящённой задаче предсказания задержек авиарейсов на основе методов машинного обучения

Целью проекта является разработка и исследование моделей машинного обучения, позволяющих прогнозировать факт задержки авиарейса на основе исторических данных о выполненных рейсах.

В рамках проекта реализован полный цикл работы с данными, включающий предварительный анализ датасета, предобработку признаков, обучение моделей классификации и оценку их качества с использованием стандартных метрик.
